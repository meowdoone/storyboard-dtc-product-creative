# DTC 镜头语言库

## 用途

让故事版图片具备可视频化的镜头逻辑。

## Camera Entries

| camera_id | 用途 | 画面方式 | 运动方式 |
|---|---|---|---|
| locked-off-demo | 清晰证明用法 | 固定机位中近景 | 动作在画面内完成 |
| top-down-tabletop | 桌面选择、开箱、定制 | 俯拍横屏 | 手部进入、对齐、移动 |
| macro-push-in | 材质、结构、细节证明 | 微距 | 缓慢推进 |
| handheld-follow | UGC 感、真实使用 | 手持轻微晃动 | 跟随商品/手部 |
| slow-dolly | 高级产品展示 | 中景到近景 | 缓慢前推或后拉 |
| rack-focus | 商品和场景关系 | 前后景切换 | 焦点从人物/道具转到商品 |
| tabletop-tracking | 桌面流程 | 横向滑动 | 跟随手部动作 |
| orbit-product-reveal | 多角度商品 | 产品居中 | 轻微环绕 |
| hero-reveal | 最终商品信心 | 产品完整出现 | 镜头稳定收束 |
| match-cut | 两个世界连接 | 相似动作/形状切换 | 动作匹配转场 |
| split-screen-comparison | 对比 | 左右分屏 | 同步展示差异 |
| over-shoulder-check | 用户确认细节 | 肩后视角 | 视线和商品同向 |

## 镜头选择规则

- 0-2s 用容易读懂的镜头，不要复杂运镜。
- 证明细节用 macro 或 locked-off。
- 使用流程用 top-down、tabletop tracking 或 handheld follow。
- End frame 用 hero reveal 或 locked-off。
- 横屏故事版避免竖屏自拍视频构图。
