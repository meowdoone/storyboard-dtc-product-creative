# DTC 场景库

## 场景选择原则

场景必须帮助商品被理解。场景不是装饰背景。

每个场景需要判断：

- 商品是否自然出现。
- 买家是否会在这个空间做购买判断。
- 商品是否能被横屏 16:9 清楚展示。
- 光线、表面、道具是否服务商品。

## Scene Lock

选择场景后，5 格必须保持：

- 同一空间
- 同一表面
- 同一光线方向
- 同一道具体系
- 同一商品比例
